este proyecto es la version beta como pudieron ver antes de leer este readme el objetivo de este proyecto es neutral e informativo que intenta aacbar con las contiendas de
los usuarios tanto de Windows como de las distribuciones de Linux entonces en este proyecto se intenta acabar con las contiendas entre los usuarios de Windows y Linux cabe
resaltar que este proyecto es una guia para aquellos que se quieran cambiar de sistema operativos o incluso probarlos ya luego de instalarlos
